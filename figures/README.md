# `figures/` — published figure PNGs

Rasterised copies of every figure that appears in the main paper and in
the supplementary material. Provided for convenience so readers can
inspect the results without running the notebooks. The figures are
regenerated by the cells in `notebook/results.ipynb` and
`notebook/annotations.ipynb` — see the *Paper figures → notebook cells*
table in the top-level `README.md`.

| File | Paper reference | Produced by |
| --- | --- | --- |
| `figure_1_semantic_map.png` | Figure 1 — semantic clustering (DistilBERT → UMAP → HDBSCAN → t-SNE, 2D + 3D) | `notebook/results.ipynb` §*Semantic map (Figure 1)* |
| `figure_2_reasons_per_technique.png` | Figure 2 — reasons occurrence ratios per technique and per technique type | `notebook/results.ipynb` §*Distribution of reasons per therapy type* |
| `figure_3_cooccurrence_heatmap.png` | Figure 3 — pairwise co-occurrence heatmap with Traditional / Alternative diagonal split | `notebook/results.ipynb` §*Distribution of reasons per therapy type* |
| `figure_4_similarity_network.png` | Figure 4 — rhetorical-proximity network (cosine ≥ 0.9, spring_layout seed 585) | `notebook/results.ipynb` §*Distribution of reasons per therapy type* |
| `figure_S1_round1_confusion_matrix.png` | Supplementary Figure S1 — round 1 IRR normalised confusion matrix | `notebook/annotations.ipynb` §*§4 Supplementary Figure S1* |
| `figure_S1_round2_confusion_matrix.png` | Supplementary Figure S1 — round 2 IRR normalised confusion matrix | `notebook/annotations.ipynb` §*§4 Supplementary Figure S1* |

These PNGs are the exact images used in the submitted manuscript. If
you rerun the notebooks against the OSF-hosted CSVs you should obtain
visually identical figures up to minor stochastic drift in the LLM
classification step (see `docs/methodology.md` §2.3).
